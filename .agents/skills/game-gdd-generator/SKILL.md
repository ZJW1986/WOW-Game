# Game GDD Generator

Generate a technical GDD that stays inside the chosen template family.

## Required Sections

1. Concept
2. Core loop
3. Player and entities
4. Level or map structure
5. Numbers and tuning
6. Implementation route

## Rules

- Do not introduce mechanics outside phase 1 template capabilities.
- Keep GDD fields machine-readable.
- Include a human-readable Markdown summary.
- Make all later asset and config decisions traceable to GDD fields.
